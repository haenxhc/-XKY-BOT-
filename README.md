⚙️ Installation

Pré-requis

- Termux ou un terminal compatible
- Node.js
- Git

Étapes

```bash
apt update && apt upgrade
pkg install git nodejs ffmpeg imagemagick yarn
termux-setup-storage
git clone https://github.com/haenxhc/-XKY-BOT-
cd -XKY-BOT-
yarn install
npm start
```
